  • Laravel

    Saya pernah mengembangkan sebuah aplikasi manajemen gudang bernama BOX yang dibangun menggunakan framework Laravel. Aplikasi ini dirancang untuk membantu pengelolaan stok barang di gudang secara efisien dan real-time. Fitur utama dari aplikasi BOX meliputi pencatatan masuk-keluar barang, manajemen kategori dan lokasi penyimpanan, notifikasi stok minimum, serta laporan stok yang dapat diekspor ke format Excel atau PDF. Dalam pengembangannya, saya memanfaatkan fitur-fitur Laravel seperti Eloquent ORM untuk relasi data, middleware untuk pengamanan akses pengguna, serta blade templating untuk tampilan antarmuka yang rapi dan user-friendly. Aplikasi ini terbukti mempermudah proses monitoring stok dan mempercepat pengambilan keputusan dalam operasional gudang.

  • Node.js

    Saya pernah mengembangkan backend untuk sebuah aplikasi mobile yang dibangun menggunakan Flutter sebagai frontend. Dalam proyek ini, saya menggunakan node.js(express.js) sebagai framework backend dan MySQL sebagai database. Saya merancang dan membangun REST API untuk berbagai fitur utama, seperti autentikasi pengguna (login, register, dan proteksi middleware), manajemen data dinamis, pengajuan cuti, serta pembaruan profil pengguna. Selain itu, saya juga memastikan integrasi yang lancar antara backend dan frontend dengan dokumentasi endpoint yang jelas dan pengujian API menggunakan Postman. Pengalaman ini memperkuat keahlian saya dalam membangun backend yang efisien, aman, dan siap diintegrasikan dengan aplikasi mobile modern.

  • Firebase

    Saya memiliki pengalaman dalam mengembangkan backend untuk aplikasi mobile berbasis Flutter, dengan keahlian khusus dalam pengelolaan database, termasuk Firebase dan MySQL. Dalam salah satu proyek, saya menggunakan Laravel untuk membangun RESTful API yang mengatur autentikasi pengguna, manajemen data profil, pengajuan cuti, serta notifikasi. Di sisi lain, saya juga mengoptimalkan penggunaan Firebase Realtime Database dan Firestore untuk kebutuhan data real-time dan sinkronisasi multi-perangkat. Integrasi antara Flutter frontend dan backend saya kelola dengan dokumentasi endpoint yang rapi serta pengujian menyeluruh menggunakan Postman. Pengalaman ini memperkuat kemampuan saya dalam membangun sistem backend yang fleksibel, aman, dan efisien, terutama pada proyek-proyek yang membutuhkan skalabilitas tinggi.
